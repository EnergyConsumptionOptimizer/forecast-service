# Forecast Service
[![License](https://img.shields.io/badge/License-Apache_2.0-blue.svg)](https://opensource.org/licenses/Apache-2.0)
![Version](https://img.shields.io/github/v/release/EnergyConsumptionOptimizer/forecast-service)

The Forecast Service is responsible for generating energy consumption predictions using ML algorithms based on historical data.
